# ENGSE203 Pre-LAB 3 — Profile Card Builder

ตัวอย่างนำก่อนทำ LAB 3 ใช้เวลาประมาณ 50–60 นาที เพื่อให้นักศึกษาเห็นวงจรพื้นฐานของ Web UI:

`Semantic HTML → Mobile-first CSS → input event → submit event → validation → feedback`

## โฟลเดอร์

- `starter/` — ไฟล์ที่มี TODO สำหรับให้นักศึกษาพิมพ์ตาม
- `solution/` — เฉลยสมบูรณ์สำหรับผู้สอน

## วิธีเปิด

เปิด `index.html` ด้วย browser ได้โดยตรง หรือใช้ VS Code Live Server ตามที่ผู้สอนกำหนด

## Checkpoints

1. Form มี `label`, `id`, `name` และลำดับ Tab ถูกต้อง
2. Mobile เป็น 1 คอลัมน์ และ Desktop เป็น 2 คอลัมน์
3. `input` event อัปเดต Live Preview
4. `submit` ใช้ `preventDefault()` และตรวจข้อมูล
5. Error แสดงใกล้ field และ success แสดงใน `role="status"`

## Deploy ตัวอย่าง

สำหรับเว็บไซต์สาธิตของผู้สอน ให้ใช้โฟลเดอร์ `week03-classroom-site/` ซึ่งมี `docs/` พร้อมเปิด GitHub Pages แบบ `main /docs`
