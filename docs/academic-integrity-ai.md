# ความซื่อสัตย์ทางวิชาการและการใช้ AI

## หลักการสำคัญ

- นักศึกษาสามารถใช้เอกสาร เว็บไซต์ ตัวอย่างโค้ด หรือ AI เพื่อช่วย **เรียนรู้และทำความเข้าใจ** ได้ตามขอบเขตที่ผู้สอนกำหนด
- นักศึกษาต้องสามารถอธิบายโค้ดที่ส่งได้ด้วยตนเอง และรับผิดชอบต่อความถูกต้องของชิ้นงาน
- ห้ามคัดลอก repository ของผู้อื่นแล้วเปลี่ยนชื่อ ห้ามส่งงานแทนกัน และห้ามใช้ token/บัญชีของผู้อื่น
- ห้าม commit secret, password, API key, cookie หรือข้อมูลส่วนบุคคลขึ้น repository

## หัวข้อที่ต้องใส่ใน README เมื่อตั้งใจใช้ AI หรือแหล่งอ้างอิง

```md
## References & AI Assistance

- Source / Documentation: <ชื่อเอกสารหรือ URL>
- AI tool used: <ชื่อเครื่องมือ>
- Used for: <ใช้ช่วยอธิบาย/ตรวจแนวคิด/ช่วยหา error>
- My adaptation: <ส่วนที่ปรับและอธิบายได้ด้วยตนเอง>
```

การเปิดเผยการใช้ AI ไม่ได้ทำให้เสียคะแนนโดยอัตโนมัติ แต่การไม่สามารถอธิบายงานของตนเอง หรือการคัดลอกผลงานโดยไม่มีการอ้างอิง อาจกระทบคะแนนและถือเป็นประเด็นความซื่อสัตย์ทางวิชาการ
