Agent Wallboard Desktop App - Icon Assets
==========================================

Required Files:
--------------
1. icon.png (256x256px) - Main application icon
2. tray-icon.png (32x32px or 16x16px) - System tray icon

Creating Icons:
--------------
Option 1: Use online tools
- https://www.favicon-generator.org/
- https://favicon.io/
- https://www.favicon.cc/

Option 2: Use image editors
- Photoshop, GIMP, or Paint.NET
- Create 256x256px PNG for main icon
- Create 32x32px PNG for tray icon

Option 3: Use placeholder (for development)
- Download from https://via.placeholder.com/256
- Save as icon.png

Recommended Design:
------------------
- Simple, recognizable symbol
- Good contrast
- Works well at small sizes
- Matches app branding

Color Scheme:
------------
Primary: #667eea (Purple)
Secondary: #764ba2 (Purple)
Background: White or transparent

Example Symbol Ideas:
--------------------
- Headset icon (for agent/call center)
- Chat bubble with user icon
- Computer monitor with person
- Communication waves
