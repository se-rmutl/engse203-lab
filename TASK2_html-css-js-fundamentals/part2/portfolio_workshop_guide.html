Git Merge Conflicts

Conflict Resolution 
# ตรวจสอบไฟล์ที่ conflict
git status

# แก้ไขไฟล์ที่ conflict (ลบ conflict markers)
# <<<<<<< HEAD
# your changes
# =======
# their changes
# >>>>>>> branch-name

# เมื่อแก้ไขเสร็จแล้ว
git add .
git commit -m "fix: Resolve merge conflicts"
2

JavaScript ไม่ทำงาน

  • ตรวจสอบ Console ใน Browser DevTools (F12)
  • ตรวจสอบว่าใส่ script tag ก่อนปิด body
  • ตรวจสอบ syntax errors
  • ตรวจสอบ DOM elements มีอยู่จริง
  • ใช้ console.log() เพื่อ debug
3

CSS Variables ไม่ทำงาน

CSS Variables Fix 
/* ตรวจสอบว่าใช้ var() function */
background-color: var(--bg-color);  /* ✅ ถูกต้อง */
background-color: --bg-color;       /* ❌ ผิด */

/* ตรวจสอบว่าประกาศ variables ใน :root */
:root {
    --bg-color: #ffffff;
    --text-color: #333333;
}
4

Animation ไม่ smooth

Performance Optimization 
/* เพิ่ม hardware acceleration */
.animated-element {
    will-change: transform, opacity;
    transform: translateZ(0);
    backface-visibility: hidden;
}

/* ใช้ transform แทน position */
/* ❌ ช้า */
left: 100px;

/* ✅ เร็ว */
transform: translateX(100px);
